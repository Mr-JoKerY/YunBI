# Yun 智能BI

## 项目介绍

一个基于AIGC的智能数据分析平台。区别于传统 BI，用户只需要导入原始数据集、并输入分析诉求，就能自动生成可视化图表及分析结论，实现数据分析的降本增效。

项目后端使用语言为Java，包括现在市面上主流技术栈，采用RabbitMQ解耦业务模块，进行图表分析，前端使用React，Ant Design Pro + Ant Design组件库，使用现成组件库快速开发项目。
